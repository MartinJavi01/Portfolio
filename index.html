Proyectos

Channel UPM

Proyecto usado como trabajo de fin de grado de ingeniería de software. Se trata de una aplicación android desarrollada en Kotlin que implementa un sistema de mensajes privados y grupales, y un sistema de noticias, con posibilidad de elección de categorías de interés para visualizar tipos de noticias concretos. La aplicación se conecta a una base de datos alojada en AWS mediante endpoints de AWSLambda desarrollados con Python.

Juego en Godot

Primera prueba de crear una demo de videojuego usando el motor de videojuegos Godot con su lenguaje de programación gdscript. Se trata de un videojuego plataformas en 2D que tiene implementaciones base de cualquier juego de este estilo: menu de inicio con opciones de audio y guardado y carga de archivos de guardado, movimiento de personaje, sistema de combate, cambio de escenas y diálogos con NPC entre otros.
